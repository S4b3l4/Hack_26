# HACKUDC 2026
GateTEXT  -->  Invocador de Documentos 🐈‍⬛🐈🐈‍⬛

Es básicamente tu centro de control personal para tener todos tus archivos en orden sin que te cueste esfuerzo. Puedes subir:
  - PDFs
  - TXTs
  - Documentos de Word
El sistema se encarga de leerlos y organizar todo por ti automáticamente. ¿Necesitas encontrar algo urgente? No te preocupes, con su buscador integrado puedes filtrar por palabras clave o fechas específicas, para que no pierdas ni un solo segundo buscando entre tanto caos. 📂✨

Hemos usado Flask para que todo corra fluido y librerías como pdfplumber y python-docx para que no se le escape nada al extraer la info. Pero lo que lo hace realmente especial es el algoritmo TextRank: básicamente, analiza el texto como un grafo para sacar las palabras clave más importantes por sí mismo. Todo esto vive en una base de datos SQLite bien estructurada que incluso cuenta con un índice integrado para mayor rapidez y optimización, así que tus documentos están siempre ahí, guardados y listos para cuando los necesites. ⚙️🧠

Si trabajas en equipo o desde varios dispositivos podrás importar o incluso eliminar de forma individual/conjunta los elementos de nuestra base de datos. Esto le aporta dinamismo y personalización. Además, tras listar los archivos es posible acceder a su contenido mediante una visualización en el propio navegador (pdf) o descargándolos (doc). 💃🎨

Por último, su gran estética, se mezcla con la nostalgia de la fuente "Press Start 2P" y el pixel art retro, crean una experiencia visual única y envolvente. Además, es totalmente responsive, así que se ve de lujo tanto en tu monitor grande como en la pantalla de tu móvil. Es la mezcla perfecta entre un sistema potente y un diseño con mucha personalidad, ideal para organizar tus archivos con estilo. 🎮
